2 — The workflow

Follow the steps

Run these steps in your AI agent (OpenCode, Claude Code, etc.).

Place the input

Copy the downloaded transcript into traNNsform/input/ (or create it if it doesn't exist).

cp better-off-ted-transcript.md traNNsform/input/

Run the workflow

Tell your agent to execute the meeting-to-summary workflow on the transcript.

Run the sample workflow meeting-to-summary on traNNsform/input/better-off-ted-transcript.md. Read the transcript, extract decisions, action items, and blockers, and produce an executive summary in traNNsform/output/. Name the file BetterOffTed_executive_summary.md.

Agent ingests the source

The agent reads the transcript, detects speaker turns and dialogue structure, and extracts metadata (source URL, duration, show name).

Agent extracts signals

The agent scans for decisions ("let's do it", "it can be fixed"), action items ("we can do anything"), and blockers ("toxins released when exposed to sunlight").

Agent writes the summary

The agent composes a structured Markdown file with: executive brief, decisions, action items table, blockers & risks, and next meeting agenda.

Review your output

Find the result at traNNsform/output/BetterOffTed_executive_summary.md. The agent will confirm and offer to adjust the level of detail.

cat traNNsform/output/BetterOffTed_executive_summary.md
3 — The result

What you'll get

The agent produces a structured executive summary like this:

📋 BetterOffTed_executive_summary.md
# Better Off Ted — Executive Summary Source: https://www.youtube.com/watch?v=NeFafCEqeh0 | Duration: 0:56 ## Executive Brief Phil presented the Viridian solar-powered oven project. While the team has made progress on a military contract to airdrop ovens as a goodwill gesture, a critical safety issue was identified: the plastic casing leeches toxins into food when exposed to sunlight — which is the only condition under which solar ovens function. ## Decisions - Proceed with fixing the toxicity issue rather than scrapping the project - The team is confident they can solve it ## Action Items | Owner | Task | Deadline | |--------|---------------------------------------------------|----------| | @phil | Investigate alternative non-toxic materials | TBD | ## Blockers - Fundamental design conflict: sunlight triggers both the oven function AND the toxin release ## Risks - Military airdrop scheduled; unresolved toxicity could cause harm - Team morale affected by recurring "everything leeches toxins" pattern ## Next Meeting - Review material alternatives and toxicity test results
