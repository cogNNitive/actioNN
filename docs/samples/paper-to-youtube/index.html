2 — The workflow

Follow the steps

Run these steps in your AI agent (OpenCode, Claude Code, etc.).

Download and place the input

Download the paper PDF and place it in traNNsform/input/.

curl -o traNNsform/input/munz2009a.pdf https://pdodds.w3.uvm.edu/files/papers/others/2009/munz2009a.pdf

Run the workflow

Tell your agent to execute the paper-to-youtube workflow on the PDF.

Run the sample workflow paper-to-youtube on traNNsform/input/munz2009a.pdf. Read the paper, identify the central question and key findings, and produce a timestamped YouTube script with visual cues in traNNsform/output/. Name the file munz2009a_youtube_script.md.

Agent ingests the PDF

The agent reads the PDF, extracts the abstract, methodology, key findings, and limitations. It identifies the central narrative arc.

Agent structures the script

The agent organizes the content into a YouTube-friendly structure: hook, context, key findings, limitations, and call to action — each with estimated timestamps.

Agent adds visual cues

The agent suggests visual elements for each segment: charts to display, text overlays, b-roll concepts, and on-screen graphics to keep viewers engaged.

Review your output

Find the result at traNNsform/output/munz2009a_youtube_script.md. The agent will confirm and offer to adjust tone or level of detail.

cat traNNsform/output/munz2009a_youtube_script.md
3 — The result

What you'll get

The agent produces a timestamped YouTube script like this:

🎬 munz2009a_youtube_script.md
# YouTube Script: What makes a video go viral? ## Hook (0:00–0:15) [Visual: Rapid montage of viral videos — cat, dance, fail compilations] "Some videos explode overnight. Others die in silence. What's the difference?" ## Intro (0:15–0:45) [Visual: Host on camera + title card] "Today we're breaking down a 2009 paper by Philip Munz that used agent-based modeling to figure out what makes a video spread." ## Finding 1: Emotional contagion (0:45–1:45) [Visual: Chart showing high-arousal emotions vs. views] "Content that triggers high-arousal emotions — awe, anger, amusement — is 3x more likely to be shared than neutral content." ## Finding 2: Network structure (1:45–2:45) [Visual: Network graph animation showing weak ties] "It's not just what you share, but who you are. Weak-tie networks spread content faster than dense friend clusters." ## Limitations (5:00–5:30) [Visual: Text overlay — "2009 data, small sample"] "The study used 2009 YouTube data and a small sample. Platform algorithms have changed dramatically since then." ## CTA (5:30–6:00) [Visual: Host on camera + subscribe overlay] "What's a video you've seen go viral recently? Drop it in the comments."
