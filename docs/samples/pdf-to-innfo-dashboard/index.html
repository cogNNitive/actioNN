Take a Wikipedia article PDF, run it through the full cogNNitive pipeline: normalize with traNNsform, model with iNNfo, and export an interactive Charts.js dashboard.

2 — The workflow

Follow the steps

Run these steps in your AI agent (OpenCode, Claude Code, etc.). This sample chains two skills: nn-trannsform for document processing and HTML export, and nn-innfo for model authoring.

Save and place the PDF

Open the Wikipedia article and save it as PDF (File → Print → Save as PDF). Place it in traNNsform/input/. Alternatively, download the source text and use it directly.

curl -o traNNsform/input/Whats_Up_Doc_1972_film.pdf "https://en.wikipedia.org/api/rest_v1/page/pdf/What%27s_Up%2C_Doc%3F_%281972_film%29"

Run the pipeline

Tell your agent to execute the pdf-to-innfo-dashboard workflow on the PDF.

Run the sample workflow workflows/samples/pdf-to-innfo-dashboard.workflow.md on traNNsform/input/Whats_Up_Doc_1972_film.pdf. Read the PDF, normalize it, create an iNNfo specialization and model, then export an interactive dashboard to traNNsform/output/. Name the model WhatsUpDoc_1972_V_0-1-0_film_NN.md and the dashboard WhatsUpDoc_1972_dashboard.html.

Agent ingests the PDF

The agent detects the PDF and uses pdf-parse (Node.js) to extract the full text. If the agent reads PDFs natively, it does so directly. The content is normalized to structured Markdown with source traceability.

Agent designs the specialization

Using nn-innfo, the agent analyzes the extracted content and identifies film-specific concepts: Film metadata, Cast & Crew, Plot beats, Production details, Awards & Reception. It proposes a level-2 specialization with typed fields and matrices.

Agent authors the iNNfo model

The agent writes the specialization (level 2) and model (level 3) files, validates them via innfo-mcp validate_model, and confirms the schema is correct. You get two validated *_NN.md files.

Agent exports the dashboard

Using nn-trannsform's export pipeline, the agent reads the model, resolves the specialization, and generates a standalone HTML dashboard with Charts.js visualizations: cast grid, budget vs. box office bar chart, awards timeline, and plot summary.

open traNNsform/output/WhatsUpDoc_1972_dashboard.html
3 — The architecture

How the pipeline fits together

Three skills working in sequence: import, model, export.

📑
Stage 1: Import
nn-trannsform — Normalize the PDF to structured Markdown via pdf-parse
🧠
Stage 2: Analyze
nn-innfo — Extract film-specific concepts from the content and design the specialization schema
📝
Stage 3: Author
nn-innfo — Write specialization + model, validate via innfo-mcp
📊
Stage 4: Export
nn-trannsform — Generate interactive HTML dashboard with Chart.js
PDF ➡️ nn-trannsform import ➡️ Normalized MD ➡️ nn-innfo analyze ➡️ Concepts ➡️ nn-innfo author ➡️ *_NN.md ➡️ nn-trannsform export ➡️ dashboard.html
📦 Artifacts produced:
  • traNNsform/md/Whats_Up_Doc_1972_normalized.md — Normalized source
  • WhatsUpDoc_1972_film_V_0-1-0_spec_NN.md — Film specialization (level 2)
  • WhatsUpDoc_1972_V_0-1-0_film_NN.md — Film model (level 3)
  • traNNsform/output/WhatsUpDoc_1972_dashboard.html — Interactive dashboard
