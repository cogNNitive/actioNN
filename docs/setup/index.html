Prerequisites

Before you begin

To run any sample, just tell your AI agent to use the bootstrap URL. No Git or terminal needed.

1

Install OpenCode Desktop

Download and install the OpenCode Desktop application from opencode.ai. Follow the instructions for your platform.

# Verify installation opencode --version
2

Tell your AI agent

Open OpenCode Desktop and tell your agent:

I want to use https://cognnitive.com

The agent will fetch the bootstrap manifest, download all skills from GitHub, register the MCP server, and present a workflow menu — everything automatic.

3

Choose & follow

Pick what you want to do from the workflow menu and follow the instructions your agent provides. No terminal, no Git, no manual setup.

✅ Verify everything works

After the agent completes bootstrap, try loading a skill:

> Load the nn-trannsform skill
# Expected: "🔧 You're using skill: nn-trannsform"

✔ Ready? Go to samples →
