# iNNv0 Design Presets

Reference material — load on demand when working on web files. Each preset in `presets/` defines a complete visual identity: palette, typography stack, spacing grid, and layout rules.

## Available presets

- [`morado-nazareno`](presets/morado-nazareno.md) — Brand primary `#4D0E4E`, strict light mode, 8px grid, Plus Jakarta Sans + Playfair Display

Browse the `presets/` directory for the full list. When a user asks about visual design, read the relevant preset and apply its tokens.
