AI Agent Skills, Engineered for OpenCode

iNNv0 es un ecosistema de skills modulares que potencian a tu agente de IA con capacidades especializadas: evaluación de modelos, gestión de skills, transformación de documentos y diseño web.

Ecosistema

Skills que transforman tu agente

Cada skill es un módulo independiente con instrucciones precisas para que el agente actúe con propósito y conocimiento de dominio.

M

Model Router

Evalúa si el modelo de IA que estás usando es el adecuado para cada tarea. Clasifica solicitudes en 4 escalas y recomienda el modelo óptimo según costo y capacidad.

Ver documentación →
S

Skills Manager

Meta-skill que administra todos los skills del repositorio. Escanea, detecta tipos de instalación (Junction, Symlink, Copy), y gestiona el ciclo de vida de cada skill.

Ver documentación →
T

traNNsform

Pipeline completo de ingestión y transformación de documentos. Convierte PDFs, DOCX, y más a Markdown unificado usando el propio LLM del agente como motor de transformación.

Sistema de diseño completo con paleta Morado Nazareno, tipografía sistemática y grid de 8px. Incluye guías para layouts comerciales y documentación técnica con Docsify.

Arquitectura

Cómo funciona

Los skills se cargan bajo demanda en el agente de OpenCode. Cada skill define su propio dominio, lenguaje y reglas de comportamiento.

Escaneo automático

Al iniciar sesión, el Skills Manager detecta todos los skills disponibles en el repositorio y verifica su estado de instalación.

Carga bajo demanda

Cuando una tarea coincide con un skill, el agente carga sus instrucciones y ejecuta el flujo de trabajo especializado.

Persistencia con Engram

Las decisiones, descubrimientos y configuraciones se guardan en memoria persistente, disponibles entre sesiones.

Primeros pasos

Instalación en segundos

Clona el repositorio y deja que el Skills Manager configure todo automáticamente con junctions de Windows NTFS.

# Clonar el repositorio git clone https://github.com/lucasventurini/iNNv0_skills.git # Iniciar OpenCode en el directorio opencode . # El Skills Manager escaneará y ofrecerá instalar todo automáticamente # Usa junctions (New-Item -ItemType Junction) para reflejar cambios en vivo
